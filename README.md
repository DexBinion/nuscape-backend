# NuScape Web UI

Dark, minimal UI for usage tracking and controls. Built with Vite + React + TS + Tailwind + Recharts.

## Quickstart
```bash
npm i
cp .env.example .env   # set VITE_API_BASE (and VITE_API_TOKEN if needed)
npm run dev
```

## Build & preview
```bash
npm run build
npm run preview
```

## Notes
- If the API is unreachable, the UI falls back to **mock data** so you can keep developing.
- Update colors/layout in `tailwind.config.ts`.
